#  Litter Detection System using Computer Vision

##  Overview
This project is an AI-based litter detection system designed to identify garbage and waste in public areas using Computer Vision techniques. It helps in maintaining cleanliness and supports smart city initiatives by detecting litter in real-time.

##  Features
- Real-time litter detection using camera input
- Detects waste materials like plastic, paper, and garbage
- Uses object detection for accurate identification
- Can be integrated with monitoring systems
- Helps automate cleanliness management

##  Tech Stack
- Python
- OpenCV
- YOLOv8
- NumPy

##  How It Works
1. Capture image or video input
2. Use YOLO model to detect litter objects
3. Highlight detected waste using bounding boxes
4. Display results in real-time
